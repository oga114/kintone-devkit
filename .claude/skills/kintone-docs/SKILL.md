# kintone Docs 生成スキル

kintoneアプリの設計情報からMarkdownドキュメントを自動生成します。

## 使用フロー

1. `/kintone-schema {APP_ID} all` でアプリ情報を取得
2. 取得した情報をMarkdown形式に整形
3. ドキュメントを出力

## 出力ドキュメント構成

```markdown
# {アプリ名} 設計書

## 基本情報
- アプリID: {id}
- 作成者: {creator}
- 作成日時: {createdAt}

## フィールド一覧
| フィールドコード | ラベル | 型 | 必須 | 備考 |
|-----------------|--------|-----|------|------|
| ... | ... | ... | ... | ... |

## ビュー一覧
| ビュー名 | 種類 | 条件 |
|----------|------|------|
| ... | ... | ... |

## フォームレイアウト
（構造図）
```

## 生成例

### 入力
```
/kintone-docs 123
```

### 出力

```markdown
# 顧客管理アプリ 設計書

## 基本情報

| 項目 | 値 |
|------|-----|
| アプリID | 123 |
| アプリ名 | 顧客管理 |
| 説明 | 顧客情報を管理するアプリ |
| 作成者 | admin |
| 作成日時 | 2026-01-15 |

---

## フィールド一覧

### 基本情報グループ

| コード | ラベル | 型 | 必須 | ユニーク | 備考 |
|--------|--------|-----|:----:|:-------:|------|
| `company_name` | 会社名 | SINGLE_LINE_TEXT | ○ | - | - |
| `contact_name` | 担当者名 | SINGLE_LINE_TEXT | ○ | - | - |
| `email` | メールアドレス | LINK | - | ○ | URL型 |
| `phone` | 電話番号 | SINGLE_LINE_TEXT | - | - | - |

### 取引情報グループ

| コード | ラベル | 型 | 必須 | ユニーク | 備考 |
|--------|--------|-----|:----:|:-------:|------|
| `status` | ステータス | DROP_DOWN | ○ | - | 選択肢: 新規/対応中/完了 |
| `amount` | 金額 | NUMBER | - | - | 単位: 円 |
| `contract_date` | 契約日 | DATE | - | - | - |

### システムフィールド

| コード | ラベル | 型 |
|--------|--------|-----|
| `$id` | レコード番号 | RECORD_NUMBER |
| `$revision` | リビジョン | __REVISION__ |
| `creator` | 作成者 | CREATOR |
| `created_time` | 作成日時 | CREATED_TIME |
| `modifier` | 更新者 | MODIFIER |
| `updated_time` | 更新日時 | UPDATED_TIME |

---

## ビュー一覧

| ビュー名 | 種類 | ソート | フィルタ条件 |
|----------|------|--------|-------------|
| (すべて) | LIST | 作成日時 降順 | - |
| 対応中 | LIST | 更新日時 降順 | `status = "対応中"` |
| 月別集計 | CALENDAR | - | - |

---

## フォームレイアウト

\`\`\`
┌─────────────────────────────────────┐
│ [基本情報]                           │
│ ┌─────────────┬─────────────┐       │
│ │ 会社名       │ 担当者名     │       │
│ ├─────────────┼─────────────┤       │
│ │ メール       │ 電話番号     │       │
│ └─────────────┴─────────────┘       │
├─────────────────────────────────────┤
│ [取引情報]                           │
│ ┌─────────────┬─────────────┐       │
│ │ ステータス   │ 金額        │       │
│ ├─────────────┴─────────────┤       │
│ │ 契約日                     │       │
│ └───────────────────────────┘       │
└─────────────────────────────────────┘
\`\`\`

---

## 関連アプリ

| 関連名 | 参照先アプリ | キーフィールド |
|--------|-------------|---------------|
| 案件情報 | 案件管理(124) | company_name |

---

*生成日時: 2026-02-03*
```

## オプション

```
/kintone-docs 123              # 全情報
/kintone-docs 123 fields       # フィールド一覧のみ
/kintone-docs 123 views        # ビュー一覧のみ
/kintone-docs 123 --output FILE # ファイル出力
```

## 出力先

生成したドキュメントは以下に保存を推奨：
- `docs/kintone/app-{APP_ID}.md`
- `docs/specifications/kintone-app-{APP_ID}.md`
